Chlorophyll-a

Usage

chlafiles(time.resolution = c("weekly", "monthly"), product = c("johnson",
  "oceancolor"), platform = c("MODISA", "SeaWiFS"), ...)

Arguments

time.resolution
weekly (8day) or monthly
product
choice of chla product, see readchla
platform
(modis[a] or seawifs)
...
reserved for future use, currently ignored

Value

data.frame

Description

Chlorophyll-a for the Southern Ocean

Details

This function generates a list of available chlorophyll-a files, including SeaWiFS and MODIS.
